CONTROL FOR AUTOMATIC VACUUM PUMPS DEPENDS ON THE ENGINE TYPE OF YOUR SHIP.

CHANGE TO ON TO ACTIVATE, OFF TO DEACTIVATE.

STELLAR TRIANGULATION MATRIX

1 - SET SLIDER 1 BASED ON THE COLOR OF LIGHT (SEE: SPACE FORCE COLOR CODE CHART)

2 - IF THE NUMBER OF LETTERS IN THE SHIP'S MODEL NUMBER IS ODD, USE BUTTON 1, OTHERWISE USE BUTTON 2.

SET THE BUTTON TO ON TO ACTIVATE, OFF TO DEACTIVATE.

JETTISON EMERGENCY PUPPIES

IN THE EVENT OF AN EMERGENCY IT MAY BE NECESSARY FOR YOU TO DUMP YOUR CARGO OF ADORABLE CUDDLY PUPPIES INTO THE COLD VACUUM OF DEEP SPACE.

1 - SET SLIDER 2 TO 4

2 - ENTER 4X6 ON THE KEYPAD. X IS THE VALUE OF JAM LEVELS

3 - SET KNOB 1 TO THE VALUE OF KNOB 2
